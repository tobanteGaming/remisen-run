## Quick Start

- C++17
- CMake > 3.13
- Ninja Build

Clone with submodules:

```sh
git clone --recursive https://github.com/tobanteGaming/remisen-run.git
```

```sh
CONFIG=Debug make
CONFIG=Release make
```
